# Référence Calendrier v0.1

Référence figée de Calendrier, application Django/React auto-hébergée issue du contrat app-template démontré. Elle prépare trois documents indépendants : guide utilisateur, guide d’exploitation et référence développeur.

La génération doit utiliser le mode `sections` : les prompts complets sont refusés. Les 38 sections sont déterministes et le contexte local prévu est de 12 288 tokens.

Les artefacts sont des intrants de benchmark reproductibles ; ils ne contiennent aucune valeur de variable sensible.
