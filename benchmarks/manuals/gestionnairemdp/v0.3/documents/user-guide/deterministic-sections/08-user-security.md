### Contrôles de sécurité démontrés

| Sujet | Consigne |
|---|---|
| Connexion | Utilisez votre compte pour accéder aux fonctions protégées. |
| Informations sensibles | Ne publiez pas d’informations sensibles dans vos échanges ou captures. |
| Données privées | Elles sont chiffrées côté client avant stockage et déchiffrées localement pendant l’utilisation. |
| Clé exportée | Considérez toute clé exportée comme une information sensible ; ne la publiez pas et conservez-la dans un emplacement approprié. |
